# 儿童安全座椅设计助手 PWA安装指南

## 应用信息

- **应用名称**：座椅助手
- **应用类型**：PWA（渐进式Web应用）
- **版本**：V7.5.0
- **主题色**：紫色系 (#667eea)

## 准备工作

1. **确保应用服务正在运行**
   - 应用已部署在开发环境
   - 访问地址：`http://localhost:5000`
   - ⚠️ **注意**：PWA安装需要HTTPS环境，但在开发时，如果你和手机在同一WiFi网络，可以使用设备的局域网IP访问

2. **连接同一WiFi网络**
   - 确保电脑和手机连接到同一个WiFi网络
   - 获取电脑的局域网IP地址

## 获取电脑的局域网IP地址

### Windows系统
1. 打开命令提示符（CMD）
2. 输入命令：`ipconfig`
3. 找到"IPv4 地址"，例如：`192.168.1.100`

### macOS系统
1. 打开终端
2. 输入命令：`ifconfig`
3. 查找`en0`部分的`inet`地址，例如：`192.168.1.100`

### Linux系统
1. 打开终端
2. 输入命令：`ip addr show`
3. 查看对应网卡的IP地址

## Android设备安装步骤

### 方法一：通过Chrome浏览器安装（推荐）

1. **打开Chrome浏览器**
   - 在手机上打开Chrome浏览器

2. **访问应用地址**
   - 在地址栏输入：`http://[你的电脑IP]:5000`
   - 例如：`http://192.168.1.100:5000`

3. **点击安装提示**
   - 浏览器会自动显示"添加到主屏幕"或"安装应用"的提示
   - 点击"安装"按钮

4. **确认安装**
   - 系统会显示应用信息
   - 点击"添加"或"安装"确认

5. **完成安装**
   - 应用图标会出现在手机主屏幕
   - 像原生应用一样打开使用

### 方法二：手动安装

如果没有自动提示：

1. 点击Chrome浏览器右上角的"⋮"菜单
2. 选择"添加到主屏幕"或"安装应用"
3. 确认安装

### 卸载方法

- 长按应用图标
- 选择"卸载"
- 确认卸载

## iOS（iPhone/iPad）设备安装步骤

### 方法一：通过Safari浏览器安装（推荐）

1. **打开Safari浏览器**
   - 在iPhone或iPad上打开Safari浏览器

2. **访问应用地址**
   - 在地址栏输入：`http://[你的电脑IP]:5000`
   - 例如：`http://192.168.1.100:5000`

3. **点击分享按钮**
   - 点击浏览器底部的"分享"图标（方形带向上箭头）

4. **添加到主屏幕**
   - 在分享菜单中向下滑动
   - 找到并点击"添加到主屏幕"

5. **确认安装**
   - 可以修改应用名称（默认为"座椅助手"）
   - 点击右上角的"添加"按钮

6. **完成安装**
   - 应用图标会出现在主屏幕
   - 点击图标即可像原生应用一样使用

### 方法二：注意事项

⚠️ **iOS特殊要求**：

1. **必须使用Safari浏览器**
   - 不能使用Chrome或其他浏览器
   - Safari是iOS上唯一支持PWA安装的浏览器

2. **iOS版本要求**
   - iOS 11.3及以上版本才支持PWA

3. **安装后使用**
   - 从主屏幕打开应用
   - 不要从Safari书签打开（这样会显示地址栏）

### 卸载方法

- 长按应用图标
- 点击"删除应用"
- 或进入设置 > 通用 > iPhone存储空间 > 找到应用并删除

## 常见问题

### Q1: 为什么无法安装？

**可能原因**：
- iOS设备使用了Chrome浏览器（必须使用Safari）
- 网络连接不稳定
- IP地址输入错误
- 电脑防火墙阻止了连接

**解决方法**：
- iOS设备务必使用Safari浏览器
- 确保手机和电脑在同一WiFi
- 检查IP地址是否正确
- 临时关闭防火墙测试

### Q2: 安装后应用无法打开？

**解决方法**：
- 确保电脑上的应用服务正在运行
- 检查电脑和手机是否在同一网络
- 重新安装应用

### Q3: 提示"不安全"怎么办？

**原因**：
- 开发环境使用HTTP而非HTTPS
- PWA生产环境应该使用HTTPS

**临时解决**：
- 在浏览器地址栏显示"不安全"警告时
- 点击"继续访问"或"高级"选项
- 仍然可以安装和使用

**正式部署**：
- 生产环境必须使用HTTPS
- 可以使用免费的SSL证书（Let's Encrypt）

### Q4: 数据会丢失吗？

**不会**：
- 应用的所有数据存储在本地
- 浏览器缓存不会影响应用数据
- 卸载应用前请导出重要数据

### Q5: 如何更新应用？

**自动更新**：
- 打开应用时会自动检查更新
- 后台自动下载新版本
- 下次启动时应用新版本

**手动更新**：
- 重新打开应用
- 等待自动更新完成
- 刷新页面（下拉刷新）

## 生产环境部署建议

如果要正式部署供所有人使用，建议：

1. **使用HTTPS**
   - 申请SSL证书
   - 使用Nginx/Apache等Web服务器
   - 配置HTTPS

2. **使用域名**
   - 注册域名
   - 配置DNS解析
   - 使用域名访问应用

3. **云服务部署**
   - 阿里云、腾讯云、AWS等
   - 使用Docker容器化
   - 配置反向代理

4. **性能优化**
   - 启用CDN加速
   - 压缩资源文件
   - 优化Service Worker缓存策略

## 技术支持

如果遇到问题，请检查：

1. **浏览器控制台**
   - 查看是否有错误信息
   - 检查Service Worker是否注册成功

2. **网络连接**
   - 检查WiFi是否稳定
   - 测试能否访问其他网站

3. **防火墙设置**
   - 确保端口5000未被阻止
   - 检查电脑防火墙规则
