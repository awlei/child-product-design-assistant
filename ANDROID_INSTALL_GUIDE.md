## 📦 方式一：Capacitor Cloud 在线打包（最简单，推荐新手）

### ⏱️ 所需时间：5-10分钟

### 前提条件
- ✅ 一台Windows、Mac或Linux电脑
- ✅ Node.js（https://nodejs.org/ 下载LTS版本安装）
- ✅ 稳定的网络连接

### 详细步骤

#### 第1步：下载项目（2分钟）

1. **下载压缩包**
   ```
   点击这个链接下载：
   https://coze-coding-project.tos.coze.site/coze_storage_7598078310014550031/child-product-design-assistant-v8.0.0-mobile.tar_57d28664.gz?sign=1769737263-bfd76c85c2-0-39dd336f272e0356a61f9d46815113cc9bd7dd280f2d6150fce2f58fec592ebd
   ```

2. **解压文件**
   - Windows：右键 → 解压到当前文件夹
   - Mac：双击解压
   - Linux：`tar -xzf child-product-design-assistant-v8.0.0-mobile.tar.gz`

3. **进入项目文件夹**
   - 双击打开 `child-product-design-assistant` 文件夹

#### 第2步：安装 Node.js（如果还没有）

1. **下载 Node.js**
   - 访问：https://nodejs.org/
   - 点击 "Download LTS"（长期支持版本）
   - Windows下载 .msi 文件
   - Mac下载 .pkg 文件

2. **安装**
   - 双击下载的文件
   - 点击"下一步"直到完成

3. **验证安装**
   - 打开命令提示符（Win + R → 输入 cmd → 回车）
   - 输入：`node -v`
   - 如果显示版本号（如 v20.x.x），说明安装成功

#### 第3步：安装依赖（1-2分钟）

1. **打开命令行**
   - Windows：在项目文件夹空白处按住 Shift + 右键 → 选择"在此处打开PowerShell窗口"
   - Mac：在项目文件夹中右键 → 服务 → 新建位于文件夹位置的终端窗口

2. **安装 pnpm**
   ```
   npm install -g pnpm
   ```

3. **安装项目依赖**
   ```
   pnpm install
   ```
   - 等待安装完成（可能需要1-2分钟）

#### 第4步：构建Web应用（2-3分钟）

```
pnpm run build
```

- 等待构建完成
- 看到 "Build completed successfully!" 表示成功

#### 第5步：注册 Capacitor Cloud（2分钟）

1. **访问网站**
   ```
   https://cloud.capacitorjs.com/
   ```

2. **创建免费账号**
   - 点击 "Sign up"
   - 使用邮箱注册（推荐Gmail）
   - 验证邮箱

3. **获取 API Key**
   - 登录后，复制你的 API Key

#### 第6步：构建Android APK（3-5分钟）

1. **登录 Capacitor Cloud**
   ```
   npx @capacitor/cloud-cli login
   ```
   - 输入邮箱和密码
   - 或选择"Sign in with Google"

2. **构建应用**
   ```
   npx cap cloud:build android
   ```
   - 等待云端构建（3-5分钟）
   - 会显示构建进度

3. **下载APK**
   - 构建完成后，访问 Capacitor Cloud 控制台
   - 找到你的构建记录
   - 点击下载 APK 文件

#### 第7步：安装到手机（1分钟）

参考下面的【通用安装方法】章节

## 📦 方式二：直接安装预构建APK（最快速）

### ⏱️ 所需时间：3-5分钟

如果你已经有预构建的APK文件，可以直接安装。

### 详细步骤

#### 第1步：获取APK文件

**方法A： Capacitor Cloud 构建**
1. 按照方式一的第1-6步完成
2. 下载构建好的APK

**方法B：找开发者获取**
- 如果有开发者已经打包好的APK，直接使用

#### 第2步：允许安装未知应用

**Android 8.0 及以上系统：**

1. **打开设置**
   - 在手机桌面找到"设置"应用并打开

2. **进入安全设置**
   - 找到"安全"或"安全与隐私"
   - 不同品牌路径略有不同：
     - 华为/荣耀：设置 → 安全和隐私 → 允许安装未知来源
     - 小米：设置 → 更多设置 → 安全中心 → 未知来源
     - OPPO/vivo：设置 → 安全 → 允许安装未知来源
     - 三星：设置 → 生物识别和安全 → 安装未知应用

3. **允许安装**
   - 找到你的浏览器（如Chrome、UC浏览器）
   - 或找到"文件管理"应用
   - 打开开关，允许"未知来源"

**Android 13 及以上：**
1. 设置 → 应用 → 特殊应用权限 → 安装未知应用
2. 选择你的文件管理器或浏览器
3. 选择"允许"

#### 第3步：安装APK

**方法A：通过浏览器下载安装**

1. **下载APK**
   - 将APK文件上传到云盘（百度网盘、Google Drive等）
   - 在手机浏览器中打开下载链接
   - 点击下载

2. **安装**
   - 下载完成后，点击APK文件
   - 点击"安装"按钮
   - 等待安装完成

**方法B：通过USB传输安装**

1. **USB连接手机**
   - 用USB线连接手机到电脑
   - 手机上选择"文件传输"（MTP）

2. **复制APK到手机**
   - 在电脑上打开"此电脑"
   - 找到你的手机
   - 将APK文件复制到"下载"文件夹

3. **在手机上安装**
   - 断开USB连接
   - 打开手机的"文件管理"应用
   - 找到"下载"文件夹
   - 点击APK文件
   - 点击"安装"

**方法C：通过WiFi传输安装**

1. **使用QQ/微信传输**
   - 在电脑上登录QQ/微信
   - 发送APK文件到"文件传输助手"
   - 在手机上接收并安装

2. **使用局域网传输工具**
   - 安装"快牙"、"茄子快传"等应用
   - 通过WiFi快速传输

#### 第4步：完成安装

1. **等待安装完成**
   - 看到提示"应用已安装"
   - 点击"打开"或"完成"

2. **首次打开**
   - 在桌面找到"儿童产品设计助手"图标
   - 点击打开
   - 同意用户协议（如有）

## 📦 方式三：本地构建APK（适合开发者）

### ⏱️ 所需时间：20-30分钟

### 前提条件
- ✅ JDK 11 或更高版本
- ✅ Android Studio
- ✅ 8GB以上内存

### 详细步骤

#### 第1步：安装JDK

1. **下载 JDK**
   ```
   https://www.oracle.com/java/technologies/downloads/
   ```
   - 选择 JDK 11 或更高版本
   - Windows 下载 .exe
   - Mac 下载 .dmg

2. **安装并配置环境变量**
   - 安装JDK
   - 设置 JAVA_HOME 环境变量

3. **验证安装**
   ```
   java -version
   ```

#### 第2步：安装 Android Studio

1. **下载**
   ```
   https://developer.android.com/studio
   ```

2. **安装**
   - 双击安装程序
   - 按照向导安装

3. **安装 SDK**
   - 打开 Android Studio
   - SDK Manager → 安装 Android SDK Platform-Tools
   - 安装 Android 13.0 (API 33) 或更高版本

4. **配置环境变量**
   - 设置 ANDROID_HOME 环境变量
   - 将 `%ANDROID_HOME%\platform-tools` 添加到 PATH

#### 第3步：构建APK

1. **下载项目并解压**

2. **打开命令行，进入项目文件夹**

3. **运行构建脚本**
   ```
   ./scripts/build-android.sh
   ```

4. **打开 Android Studio**
   ```
   npx cap open android
   ```

5. **在Android Studio中构建**
   - 等待Gradle同步完成
   - 点击菜单 Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 等待构建完成

6. **找到APK**
   - 在通知栏点击 "locate"
   - 或手动找到：`android/app/build/outputs/apk/debug/app-debug.apk`

7. **安装到手机**
   - 参考【通用安装方法】

## 🔧 通用安装问题解决

### 问题1：无法安装，提示"解析包错误"

**原因：**
- APK文件损坏
- 下载不完整

**解决方法：**
1. 重新下载APK文件
2. 确保下载完成后文件大小正确
3. 尝试通过USB传输安装

### 问题2：提示"已拦截安装"

**原因：**
- 系统禁止安装未知来源应用

**解决方法：**
1. 按照上面的【允许安装未知应用】步骤操作
2. 检查是否允许了正确的应用（文件管理器或浏览器）
3. Android 13需要在设置中单独授权

### 问题3：提示"应用未签名"

**原因：**
- APK签名问题

**解决方法：**
1. 确保下载的是完整构建版本
2. 使用 Capacitor Cloud 构建会自动签名
3. 或使用debug keystore手动签名

### 问题4：安装后无法打开

**原因：**
- Android版本过低
- 权限问题

**解决方法：**
1. 检查Android版本（需要7.0+）
2. 清除应用数据：设置 → 应用 → 找到应用 → 存储 → 清除数据
3. 卸载重新安装

### 问题5：安装后闪退

**原因：**
- 兼容性问题
- 缓存问题

**解决方法：**
1. 重启手机
2. 清除缓存和数据
3. 更新Android系统
4. 联系开发者反馈

## 📱 不同品牌手机的特殊设置

### 华为/荣耀

**开启安装未知来源：**
```
设置 → 安全和隐私 → 允许安装未知来源
```

**智能助手提醒：**
```
设置 → 智慧助手 → 智识搜索 → 应用安装 → 关闭
```

### 小米/Redmi

**开启MIUI优化：**
```
设置 → 更多设置 → 开发者选项 → MIUI优化 → 关闭
```

**纯净模式：**
```
设置 → 我的设备 → MIUI版本 → 连续点击7次 → 纯净模式 → 退出
```

### OPPO/vivo

**安全检查：**
```
设置 → 安全 → 允许安装未知应用
```

**OPPO安装管理：**
```
设置 → 应用管理 → 安装未知应用
```

### 三星

**智能管理：**
```
设置 → 应用 → 智能管理 → 应用权限 → 安装未知应用
```
